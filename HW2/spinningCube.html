CSE 470 HW#2: Spinning Cubes

Author: Taylor Collins        Date: February 2020
Description:
This program allows user to add a rotating cube to canvas.
Functionality:
Pressing the Rotate X,Rotate Y,Rotate Z buttons changes the axis of rotation to the X,Y or Z axis respectively for the cubes.
Rotate random axis, sets the values in the rotation function to a random value for x, y, and z.
The slider scales the cubes to a certain percentage of the model.
